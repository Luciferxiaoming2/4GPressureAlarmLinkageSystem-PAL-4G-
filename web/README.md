# PAL 4G Web

## 启动

```bash
npm install
npm run dev
```

默认开发地址为 `http://127.0.0.1:5173`，并通过 Vite 代理把 `/api` 请求转发到 `http://127.0.0.1:8001`。

## 构建

```bash
npm run build
```

## 环境变量

参考 `.env.example`：

```env
VITE_API_BASE=/api/v1
```

## 当前首期范围

- 登录鉴权与路由守卫
- 仪表盘总览、图表、近期报警和近期指令
- 报警记录分页与筛选
- 设备列表、设备详情、模块继电器手动控制
- 个人设置中的修改密码

## 联调说明

- 登录接口使用表单提交：`POST /api/v1/auth/login`
- 开发环境默认依赖后端运行在 `127.0.0.1:8001`
- 仪表盘和设备详情按文档约定每 30 秒轮询刷新一次
